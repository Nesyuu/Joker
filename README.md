# Joker Hint

A mobile-friendly party card game where players join a room, receive secret cards, and try to find the Joker.

## Run locally

```bash
npm start
```

Then open `http://localhost:4173`.

## Deploy on Render

- Create a free Render account.
- Create a new Web Service from this project.
- Use `npm install` as the build command.
- Use `npm start` as the start command.
